
EXPORTTEXT : export text from an ODS file (openOffice Document Sheet)

usage:
	java ExportText -d <outDir> -h <headerDir> -e <encoding> file.ods

	outDir    	: directory to output text data
	headerDir	: directory to output header for the text
	encoding	: encoding to use for the text (recommand either UFT-8 or ISO-8859-1)
	file.ods 	: ods file to parse

ods format: check example.ods to see the format of the ods file
