
# Requiremets for using FontDataGenerator

Microsoft .Net Framework 2.0 이 설치 되어 있어야 하며,
위의 패키지는 아래의 경로에서 다운 받으실 수 있습니다.
http://www.microsoft.com/downloads/details.aspx?displaylang=ko&FamilyID=0856eacb-4362-4b0d-8edd-aab15c5e04f5

# 폴더 구조
data : FontData.java의 원본파일이 있습니다.
doc : 관련 문서와 동영상이 있습니다.
sample : 두개의 샘플 폰트가 있습니다.
source : font.dat 를 단말에서 사용하기 위한 java 소스가 있습니다.


# 자세한 사용방법은 doc폴더의 manual 을 참고하세요.
